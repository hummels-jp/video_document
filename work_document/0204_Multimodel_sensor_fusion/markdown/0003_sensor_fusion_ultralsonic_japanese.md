- 超音波は機械波の一種であり、空気いを媒体として伝播し、減衰が早い。
- 超音波RadarのFOV（視野角）は一般的に80度から120度であり、探知距離は10メートル未満である。
- 超音波Radarは距離の測定が可能だが、位置の特定は出来ない。
- 空気中の超音波の伝播速度は温度の影響を受ける。V = V0 + 0.607*T（V0 = 332m/s）;V0は温度が0度の時の超音波の伝播速度。
